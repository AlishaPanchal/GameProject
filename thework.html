Fiat Justitia: The Work

Welcome to "The Work" page, players, where we show you a collection of the components we made for Fiat Justitia. Consider this a gallery walk. Enjoy!

Component #1: The Game Box

The Game Box is where all of the components of Fiat Justitia are stored. The front has the Fiat Justitia logo, which was printed using the Silk Screening technique, and red paint to indicate blood. For creative purposes, we, the co-creators, also put our handprints in red paint as well. The one on the left is Caitlin's, while the one on the right is Alisha's. The thumbprints are also Caitlin's.

Component #2: The Game Board

The Game Board is a visual top view of a typical courtroom. It consists of the judge's table, the jury box, the transcriber's desk, the witness stand, three counsel tables (Left, Middle & Right), and the public seating area. There are also multiple sets of doors around the room. The Game Board can be used to visualize where the bailiff is murdered, each player's position, and overall, what a courtroom looks like in real life.
